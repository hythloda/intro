Allocation groups are reviewed and released weekly.

3. Check Your Group

Use this page to review the latest public group and expand older weekly releases.

What this page shows

  • Company: the requesting validator or organization.
  • PartyID Hint: a hint or identifier associated with the request when supplied.
  • Reason: the app, product, or sponsor context for the request.
  • Weekly Release: requests are organized into weekly groups so the latest release and prior releases can be reviewed separately.
